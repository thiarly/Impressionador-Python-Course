# Impressionador-Python-Course

Este repositório contém os exercícios e projetos desenvolvidos durante o "Curso Hashtag Python Impressionador", um curso completo com 659 aulas e 190 horas de conteúdo. O curso abrange desde os conceitos básicos até projetos avançados em Python, incluindo integrações, análise de dados, automação, criação de jogos e muito mais.

## Motivação

Sou um Analista de Infraestrutura de TI, formado em Redes de Computadores e pós-graduado em Segurança da Informação. Com experiência em Linux, Zabbix, VMware e Python básico, senti a necessidade de aprofundar meus conhecimentos em programação para automatizar processos e desenvolver projetos pessoais.

Um dos principais objetivos deste curso é a criação de um sistema financeiro para gerenciar minha carteira de ações e investimentos, incluindo acompanhamento de cotações, rentabilidade, dividendos, splits, bonificações, histórico de movimentações e gráficos. Ao final, pretendo hospedar esse sistema em uma página web.

Além disso, busco oportunidades de emprego ou trabalho como freelancer como programador, combinando minha experiência em infraestrutura de TI com as habilidades de programação adquiridas neste curso.
